<h1>📧 Email Spam Detection System</h1>

<p>
An Email Spam Detection System built using <b>Machine Learning</b> and 
<b>Python</b> that classifies emails/messages as <b>Spam</b> or 
<b>Not Spam (Ham)</b>.
The project uses Natural Language Processing (NLP) techniques for text preprocessing 
and a trained ML model for accurate prediction.
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
  <li>Detects spam and non-spam emails/messages</li>
  <li>Text preprocessing using NLP</li>
  <li>Machine Learning model training and prediction</li>
  <li>User-friendly interface</li>
  <li>Fast and accurate classification</li>
  <li>Supports custom email/message input</li>
</ul>

<hr>

<h2>🛠️ Technologies Used</h2>

<ul>
  <li>Python</li>
  <li>Pandas</li>
  <li>NumPy</li>
  <li>Scikit-learn</li>
  <li>NLTK</li>
  <li>Streamlit / Flask</li>
  <li>Machine Learning</li>
  <li>Natural Language Processing (NLP)</li>
</ul>

<hr>




<h2>⚙️ How It Works</h2>

<ol>
  <li>Collects email/message dataset</li>
  <li>Cleans and preprocesses text data</li>
  <li>Converts text into numerical vectors using TF-IDF or CountVectorizer</li>
  <li>Trains a Machine Learning model</li>
  <li>Predicts whether the message is Spam or Ham</li>
</ol>

<hr>

<h2>🧠 Machine Learning Algorithms</h2>

<ul>
  <li>Naive Bayes</li>
  <li>Logistic Regression</li>
  <li>Support Vector Machine (Optional)</li>
</ul>

<hr>

<h2>📊 Dataset</h2>

<p>
The dataset contains labeled email/messages categorized as:
</p>

<ul>
  <li><b>Spam</b></li>
  <li><b>Ham (Not Spam)</b></li>
</ul>
